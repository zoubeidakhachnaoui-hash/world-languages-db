# قاعدة بيانات لغات العالم (ISO 639)

تحتوي هذه الحزمة على قاعدة بيانات شاملة لجميع لغات العالم المسجلة وفق معيار ISO 639، بما في ذلك الرموز المكونة من حرفين (ISO 639-1) والرموز المكونة من ثلاثة أحرف (ISO 639-3).

## المحتويات
- `languages_database.csv`: ملف CSV مناسب للجداول.
- `languages_database.json`: ملف JSON مناسب للمبرمجين وتطبيقات الويب.
- `languages_database.db`: قاعدة بيانات SQLite جاهزة للاستخدام في التطبيقات.
- `languages_database.xlsx`: ملف Excel للمستخدمين العاديين.

## هيكل البيانات
| الحقل | الوصف | مثال |
| :--- | :--- | :--- |
| `iso639_3` | الرمز الثلاثي (المعيار الأشمل) | `eng`, `ara` |
| `iso639_1` | الرمز الثنائي (للغات الرئيسية) | `en`, `ar` |
| `name_en` | اسم اللغة بالإنجليزية | `English`, `Arabic` |
| `name_native` | اسم اللغة الأصلي | `العربية` |
| `family` | العائلة اللغوية | `Afro-Asiatic` |
| `scope` | نطاق اللغة (فردية، كلية، إلخ) | `I` (Individual) |
| `type` | نوع اللغة (حية، قديمة، إلخ) | `L` (Living) |

## إحصائيات
- إجمالي اللغات: 7,929 لغة.
- اللغات التي تملك اختصاراً ثنائياً (ISO 639-1): 184 لغة رئيسية.
